挺坑的，题目有点没有讲清楚。大概意思是你需要按照 ABCDEFGHIJKLMNOPQRSTUVWXYZ 的顺序输入字符。我们直接哪一个数组记录每个字母出现的位置就可以了。 代码 123456789101112131415#include<bits/stdc++.h>using namespace std;int main() { int ans = 0, a[30]; string s; cin >> s; for (int i = 0; i < s.size(); i++) a[s[i] - 'A'] = i; for (int i = 0; i < s.size() - 1; i++) ans += abs(a[i] - a[i + 1]); cout << ans; return 0;}
