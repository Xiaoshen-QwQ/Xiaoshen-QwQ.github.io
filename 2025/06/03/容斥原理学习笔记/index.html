容斥原理学习笔记

容斥原理介绍

在计数时，必须注意没有重复，没有遗漏。为了使重叠部分不被重复计算，人们研究出一种新的计数方法，这种方法的基本思想是：先不考虑重叠的情况，把包含于某内容中的所有对象的数目先计算出来，然后再把计数时重复计算的数目排斥出去，使得计算的结果既无遗漏又无重复，这种计数的方法称为容斥原理。
——摘自百度百科

我们先引入一个很简单的例子，一个班里有 $50$ 个人喜欢语文，有 $42$ 个人喜欢数学，既喜欢语文的又喜欢数学的有 $12$ 个人，那么这个班里有多少个人（确保每个人都至少选了一门学科）。

很明显，这个班里有 $50 + 42 - 12 = 80$ 个人（人有点小多…）。

在举个例子，现在有玩偶 A、B、C，假设一个班喜欢 A 的有 $10$，喜欢 B 的有 $15$ 个，喜欢 C 的有 $23$ 个，喜欢 AB 的有 $6$ 个，喜欢 AC 的有 $8$ 个，喜欢 BC 的有 $12$ 个，喜欢 ABC 的 有 $2$ 个，现在班里有多少个人？

现在我们依然要减去两个都喜欢的人，但这就会导致每个种类会被多减一次，所以我们还要加上喜欢 ABC 的人，也就是这个班上有 $10 + 15 + 23 - 6 - 8 - 12 + 2 = 24$ 人。

我们现在把这个东西看成集合的形式，也就是我们假设现在喜欢 A、B、C 的人分别是集合 $A, B, C$，我们要求这个班里有多少个人就是在求 $| A \cup B \cup C |$。

根据上面的例子，那么这个式子就等于 $|A| + |B| + |C| - |A \cap B| - |B \cap C| - |A \cap C| + |A \cap B \cap C|$。

我们再将 $4$ 个集合的式子列出来，也就是 $|A \cup B \cup C \cup D| = |A| + |B| + |C| + |D| - \sum \text{2个组合的交集} + \sum \text{3个组合的交集} - \sum \text{4个组合的交集}$。

那如果我们现在要求 $n$ 个集合的并呢？

通过观察规律，我们发现 $|S_1 \cup S_2 \cup S_3 \cup \dots \cup S_n| = \sum \text{1个组合的交集} + \sum \text{2个组合的交集} + \dots + \sum \text{n个组合的交集}$ 。

错排问题

错排问题是容斥原理一个很经典的应用。

错排问题大概意思是，现在设你第 $i$ 个位置排列的是数字 $P_i$，现在要求 $P_i \not= i$，问有多少组这种排列？

我们还是从 $3$ 个元素的情况入手。

那么根据问题的要求，限制条件就是 $P_1 \not= 1, P_2 \not= 2, P_3 \not = 3$。

但我们发现这个不等号非常不好考虑，所以我们就考虑他们相等的情况。

我们用 $?$ 表示没有限制，那么这个问题的求解式就是 $(? - P_1 = 1) \cp (? - P_2 = 2) \cp (? - P_3 = 3)$。

我们根据容斥原理，那么 $? = 3! = 6$，满足一个条件的有 $3 \cp 2! = 6$，满足两个条件的 $3 \cp 1! = 2$，满足三个条件的 $1 \cp 0! = 1$，那么 $3$ 个元素就有 $6 - 6 + 3 - 1 = 2$ 种情况。

同样 $4$ 个也是一样的，一个条件也不满足的有 $4! = 24$，满足一个条件的有 $4 \cp 3! = 24$，满足两个条件的 $\binom 4 2 \cp 2! = 12$，满足三个条件的 $\binom 4 3 \cp 1! = 4$，满足四个条件的 $\binom 4 4 \cp 0! = 1$，那么 $4$ 个元素就有 $24 - 24 + 12 - 4 + 2 = 10$ 种情况。

现在推广到 $n$ 个的情况，那么求解式就是：
$$
\sum_{i = 0}^{n} = \binom n i \times (n - i)! \cp (-1)^i
$$
