A

提交情况

题目大意

有 $n$ 枚硬币，其中 $k$ 枚正面朝下，但你不知道是哪 $k$ 枚。你可以翻转硬币，问如何操作能将硬币分成两组，让两组硬币正面朝上的一样多。即对于任意的硬币正反面情况，按照你的操作都可以满足要求。

数据范围

  • $1 \le n \le 10^4$

  • $0 \le k \le n$

思路

构造方案：

第一堆选 $k$ 枚硬币，第二堆选 $n - k$ 枚硬币，且全部翻转。

证明：

设第一堆有 $a$ 枚硬币，其中有 $x$ 枚硬币正面朝上，则第二堆就有 $n - a$ 枚硬币，其中 $n - k - x$ 枚硬币正面朝上。

取 $k = a$，则第二堆有 $n - k$ 枚硬币，其中 $k - x$ 枚硬币正面朝上。将其翻转，则有 $x$ 枚硬币朝上。

B

提交情况

题面大意

你需要给出一个 $1 \sim n$ 的排列，这个排列需要满足 $\sum_{i = 1}^{n - 1} gcd(a_i + 1, a_{i + 1})$ 这个值最小。

思路

要让其最小，肯定是让他所有的 $gcd$ 求解的答案都是 $1$。

要让两项的 $gcd$ 为 $1$，容易想到这两项可以是相邻的两个数，因为任意相邻的两个 $gcd$ 的值绝对为 $1$。

所以这道题的构造方案是从小到大排列所有偶数，再从小到大排列所有奇数。

提交情况

题目大意

构造一个数列，使得这个数列的 $P_{50}$，$P_{95}$ 和 $P_{99} $为给定值。其中 $P_x$ 的意义为：数列中恰好有 $x%$ 的数小于等于这个值。

数据范围

  • $1 \le a < b < c < 10^9$

思路

这道题非常简单，有与题目中保证了 $a < b < c$，于是我们可以构造一个序列，其中有 $50$ 个 $a$，$45$ 个 $b$，$4$ 个 $c$ 和 $1$ 个 $c + 1$。

题目大意

构造一个 $n × m$ 的矩阵，使得任意“相邻两个位置的数字的和”都互不相等。

证明：

下面证明这种构造方案的正确性。

首先，任意相邻的两个数字，都必然位于两个相邻的副对角线上。考虑任意两对相邻数字的和，假设它们所在的副对角线分别是 $(a, a + 1)$ 和 $(b, b + 1)$ 。

  • 假如 $a \not= b$ ，不妨设 $a < b$ ，则第 $a$ 个副对角线上的数字小于第 $b$ 个副对角线，第 $a + 1$ 个的也小于第 $b + 1$ 个的。因此这两对和必然不可能相等。

  • 假如 $a = b$ ，同一对副对角线上的和自右上到左下是单调递增的，因此也不可能相等。 综上，这种构造方法不会产生相同的和。
