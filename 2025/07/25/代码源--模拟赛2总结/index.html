A

题目大意

水果店里有散装苹果和苹果礼盒，其中散装苹果 $a$ 元一个，苹果礼盒 $b$ 元一盒，每盒包含 $m$ 个苹果，问买至少 $n$ 个苹果最少需要多少钱。

数据范围

  • $1 \le n, m, a, b \le 1000$

思路

这道题我用了 5 发才过。

这道题可以用数学方法直接计算出来。但是我就是写不对，错了 4 发。

但注意看数据范围，$n \le 1000$，这么小的数据可以直接暴力枚举啊！

枚举卖苹果礼盒的单个苹果的个数，然后一算不就完事了吗？

这个事情告诉我们，能用简单的方法过一道题就用哪个简单的方法过。

B

题目大意

平方年是指年份为完全平方数的年，如 $2025$ 年为平方年（$2025=45^2$），现给定一个日期，问该日期离下一个平方年的 $1$ 月 $1$ 日还有多少天。

数据范围

  • $1 \le a \le 10^9$

  • $1 \le b \le 12$

  • $1 \le c \le 31$

思路

题目是要求平方年，我们可以轻而易举的求出下一个平方年在什么时候，那如何计算有多少天呢？

我们看到数据范围，$a$ 最大为 $10 ^ 9$ 。如果 $a = 10^9$，那么下一个年份就是 $1001^2$ ，我们发现这两个年份就差了 $1001^2 - 1000^2 = 2001$ 年，这么小的数字，我们就可以直接暴力枚举每个年份是闰年还是平年，只需要单独对当前年份进行处理即可。

C

题目大意

做一桌满汉全席需要 $n$ 种食材，其中第 $i$ 种食材需要 $a_i$ 克。现在蜗蜗手上第 $i$ 种食材有 $b_i$ 克，并且他有 $k$ 元钱可以购买食材，已知每种食材的价格都是一元一克，蜗蜗想知道他最多能做多少桌满汉全席。

数据范围

  • $1 \le n \le 10^5$

  • $1 \le a_i \le 10^9$

  • $0 \le b_i \le 10^9$

  • $0 \le k \le 10^{18}$

思路

很明显，答案具有单调性，如果办少的不能成满汉全席，多的也就更不能办了，于是我们可以用二分答案来解决这道题。

看到数据范围，$k$ 有 $10^{18}$，很明显，需要开 long long，那么还有一个问题，二分答案的上届呢？

这道题可以就只有一种食材（$n = 1$），然后第 $i$ 种食材只需要 $1$ 克就行（$a_i = 1$），然后拥有 $10^9$ 克这样的食材（$b_i = 10^9$），然后我有 $10^{18}$ 元钱，那么我可以办 $10^9 + 10^{18}$ 这么多桌满汉全席。

所以上界是 $10^9 + 10^{18}$ 这么大。
