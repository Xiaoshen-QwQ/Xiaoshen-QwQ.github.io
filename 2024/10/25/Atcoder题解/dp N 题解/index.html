DP N 题解

发表于2024-10-25|更新于2024-10-25|Atcoder题解
|浏览量:

题意

你有 \(n\) 堆石子，你每次可以合并相邻的石子，你要付出 \(a_i + a_{i + 1}\) 代价，求将 \(n\) 堆石子合并成一堆的最小代价。 ## 思路

定义状态：\(dp_{l, r}\) 表示从 \(l\) 到 \(r\) 合并成一堆需要付出的最小代价。

状态转移方程： \[ dp_{l, r} = \min_{k = l}^{r - 1} (dp_{l, k} + dp_{k + 1, r}) + \sum_{i = l}^{r} a_i \]

我们可以将后面的 \(\sum_{i = l}^{r} a_i\) 做一个前缀和预处理，时间复杂度为 \(\mathcal O (n ^ 3)\)。
